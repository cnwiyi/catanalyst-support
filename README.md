# Catanalyst Support

**Catanalyst** is a Catan companion app for iOS featuring a board generator, score tracker, dice tracker and quick rules guide.

## Contact & Support
For support, feedback or bug reports please email:
support@esgilsoftware.io

## App Features
- Board Generator with Random, Balanced and Optimized modes
- 1-4 and 5-6 player board support
- Score Tracker with game history
- Dice Tracker with manual input and statistics
- Quick Rules & Guide reference
